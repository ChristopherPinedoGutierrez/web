# Christopher Pinedo - Portafolio Web

## Descripción

Aplicación web personal desarrollada como una Single Page Application (SPA) que funciona como portafolio profesional y currículum digital interactivo. El proyecto permite mostrar información personal, proyectos realizados, experiencia laboral y habilidades técnicas de manera organizada y atractiva.

## Propósito

Esta aplicación tiene como objetivo principal servir como una plataforma digital para presentar mi perfil profesional como desarrollador frontend, incluyendo:

- Información personal y de contacto
- Catálogo de proyectos con detalles técnicos
- Historial de experiencia laboral con visualizaciones interactivas
- Habilidades técnicas organizadas por categorías y estados de aprendizaje

## Tecnologías Utilizadas

- **React 18** - Biblioteca principal para la interfaz de usuario
- **Material-UI (MUI)** - Framework de componentes y sistema de diseño
- **React Router DOM** - Manejo de rutas y navegación SPA
- **Recharts** - Biblioteca para gráficos y visualizaciones de datos
- **React Icons** - Iconos adicionales para tecnologías
- **Create React App** - Herramienta de construcción y desarrollo
